o HTML (Hypertext Markup Language) é a base para a criação de páginas web. Utilizando tags como <h1> a <h6> para cabeçãlhos e <p> para parágrafos, ele organiza o conteúdo de forma estruturada. Além disso, o CSS é responsável pela parte visual das páginas, como cores, fontes e disposição dos elementos. Com o uso de design responsivo, as páginas se adaptam automaticamente ao tamahho da tela de dispositivos móveis. Por ecemplo, a fórmula da água H2O e a equação de Eisten, E = mc2, utilizam subíndices e superindices para representações científicas. A evolução da web trouce a necessidade de criar páginas dinâmicas e interativas, com JavaScript. Contudo, algumas práticas antigas, como a criação de páginas estáticas, estão sendo antigas, como a criação de páginas estáticas, estão sendo descontinuadas em favor de abordagens mais modernas.
